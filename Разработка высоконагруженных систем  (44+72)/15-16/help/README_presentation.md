# Веб-презентация "Брокеры сообщений"

## Файлы

- `message_brokers_web.html` - интерактивная веб-презентация (начало)
- `message_brokers_presentation.md` - полный контент в Markdown

## Как открыть

1. Откройте `message_brokers_web.html` в браузере
2. Используйте стрелки для навигации
3. Нажмите `F` для полноэкранного режима
4. Нажмите `S` для заметок докладчика

## Структура

Презентация содержит 34 слайда:
- Слайды 1-4: Введение, сферы применения, сравнение
- Слайды 5-10: Redis
- Слайды 11-17: RabbitMQ  
- Слайды 18-24: Kafka
- Слайды 25-30: Сравнение, паттерны, best practices
- Слайды 31-34: Celery, заключение

## Примечание

Для полной работы презентации нужны файлы reveal.js из папки `../13-14/css/` и `../13-14/js/`
Если их нет, скачайте reveal.js: https://revealjs.com/
