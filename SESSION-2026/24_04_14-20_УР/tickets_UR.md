# 🎓 Билеты для зачёта по дисциплине

## **1. Основы проектирования бэкенд-архитектуры (инфраструктура Python)**

**Практика**
Спроектируйте и реализуйте backend-приложение на FastAPI с разделением на слои.

Добавьте работу с базой данных (PostgreSQL или SQLite) через ORM (SQLAlchemy или аналог). Реализуйте базовую бизнес-логику и валидацию данных.

Настройте конфигурацию через переменные окружения (.env).

**Результат:**

* структура проекта (слоистая архитектура)
* CRUD-эндпоинты (минимум 3 сущности или 3 операции)
* подключение к базе данных
* конфигурация через .env
* краткое описание архитектуры (5–7 предложений)

## **2. Системный дизайн как новый тренд в разработке**

**Практика**
Спроектируйте архитектуру системы для одного из кейсов:

* сервис сокращения ссылок
* чат-приложение
* новостная лента
* файловое хранилище

Опишите компоненты системы и взаимодействие между ними.

**Результат:**

* схема архитектуры (диаграмма или текст)
* описание:

  * API
  * база данных
  * кэш
  * очереди (если применимо)
* ответы:

  * масштабирование
  * узкие места
  * отказоустойчивость
* краткое описание (7–10 предложений)

## **3. Метрики проектирования высоконагруженных систем**

**Практика**
Реализуйте и разверните FastAPI-приложение (2–3 эндпоинта) с использованием Gunicorn + Uvicorn workers в Docker.

Обеспечьте обработку нагрузки до ~1000 RPS и проведите нагрузочное тестирование (ab / k6 / wrk / locust).

**Результат:**

* Dockerfile и/или docker-compose.yml
* конфигурация (workers, threads)
* отчёт:

  * RPS
  * latency (avg / p95)
  * ошибки
* результаты тестирования (логи/скриншоты)
