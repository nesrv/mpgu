# ⚡ Быстрая шпаргалка: Cline + Polza.ai

## 🚀 Настройка за 5 минут

### 1️⃣ Получите API ключ
1. Зайдите на **https://polza.ai**
2. Зарегистрируйтесь и пополните баланс (минимум 500₽)
3. Создайте API ключ для **Claude 3.5 Sonnet**

### 2️⃣ Настройте Cline
1. Откройте VS Code
2. Кликните на иконку Cline → Настройки (⚙️)
3. Укажите:
   - **API Provider:** `Anthropic`
   - **API Key:** `polza_ваш_ключ`
   - **Base URL:** `https://api.polza.ai/v1`

### 3️⃣ Проверьте
Напишите: `"Привет, ты работаешь?"` ✅

## 🎯 Правила эффективной работы

### ✅ Делайте так:
- Будьте конкретны: "Создай модель User с полями email, username"
- Указывайте технологии: "Используй SQLAlchemy 2.0"
- Разбивайте задачи: сначала модели, потом API, потом тесты

### ❌ Не делайте так:
- Неясные запросы: "Сделай что-нибудь"
- Слишком общие: "Создай приложение"
- Всё сразу: "Создай полный проект с авторизацией, API, тестами, деплоем..."

## 🆘 Частые проблемы

### "Invalid API Key"
→ Проверьте ключ и баланс на polza.ai

### "Rate limit exceeded"
→ Подождите 1-2 минуты или пополните баланс

### Высокий расход токенов
→ Делайте более конкретные запросы

### Cline не отвечает
→ Проверьте настройки API Provider и Base URL
