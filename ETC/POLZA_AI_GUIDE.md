# 🇷🇺 Polza.ai - Руководство по использованию

## Что такое Polza.ai?

**Polza.ai** — это российский сервис, предоставляющий доступ к различным AI-моделям, включая Claude от Anthropic. Это особенно актуально для пользователей из России, где прямой доступ к некоторым AI-сервисам ограничен.

## 🌟 Основные преимущества

### ✅ Для российских пользователей:
- 🇷🇺 Работает из России без VPN
- 💳 Оплата российскими картами
- 🔄 Единый API для разных моделей
- 💰 Гибкие тарифы
- 🛡️ Соответствие российскому законодательству

### 🤖 Доступные модели:
- **Claude 3.5 Sonnet** (Anthropic) - для Cline
- **GPT-4** (OpenAI)
- **YandexGPT**
- **GigaChat** (Сбер)
- И другие модели

## 🚀 Быстрый старт с Polza.ai

### Шаг 1: Регистрация

1. Перейдите на **https://polza.ai**
2. Нажмите "Регистрация" или "Войти"
3. Зарегистрируйтесь через:
   - Email
   - Telegram
   - Яндекс ID
   - VK

### Шаг 2: Пополнение баланса

1. Войдите в личный кабинет
2. Перейдите в раздел "Баланс" или "Пополнить"
3. Выберите сумму (минимум обычно 100-500 рублей)
4. Оплатите удобным способом:
   - 💳 Банковская карта (Visa, MasterCard, МИР)
   - 📱 СБП (Система быстрых платежей)
   - 💰 ЮMoney
   - 🏦 Другие способы

### Шаг 3: Получение API ключа

1. В личном кабинете найдите раздел **"API ключи"** или **"Настройки"**
2. Нажмите **"Создать новый ключ"**
3. Укажите название (например, "Cline VSCode")
4. Выберите модель: **Claude 3.5 Sonnet**
5. Скопируйте полученный ключ (он будет виден только один раз!)

**Пример ключа:**
```
polza_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Шаг 4: Настройка Cline в VS Code

#### Вариант A: Через настройки расширения

1. Откройте VS Code
2. Нажмите на иконку Cline в боковой панели
3. Нажмите на иконку настроек (⚙️)
4. Выберите **"API Provider"**: `Anthropic`
5. В поле **"API Key"** вставьте ключ от Polza.ai
6. В поле **"Base URL"** (если есть) укажите: `https://api.polza.ai/v1`

#### Вариант B: Через файл настроек

1. Откройте настройки VS Code (`Ctrl+,` или `Cmd+,`)
2. Найдите настройки Cline
3. Добавьте:
```json
{
  "cline.apiProvider": "anthropic",
  "cline.apiKey": "polza_ваш_ключ_здесь",
  "cline.baseUrl": "https://api.polza.ai/v1"
}
```

### Шаг 5: Проверка работы

1. Откройте Cline в VS Code
2. Напишите простой запрос: `"Привет, ты работаешь?"`
3. Если получили ответ — всё настроено правильно! ✅

### Что такое токены?

**Токен** ≈ 0.75 слова (для английского) или ≈ 0.5 слова (для русского)

**Примеры расхода:**
- Простой вопрос-ответ: ~500-1000 токенов (1-3₽)
- Создание файла с кодом: ~2000-5000 токенов (5-15₽)
- Большой рефакторинг: ~10000-20000 токенов (30-60₽)

### Рекомендации по бюджету:

- **Для начала:** 500₽ (хватит на несколько дней активной работы)

## 🔧 Настройка для разных инструментов

### Для Cline (VS Code)
```json
{
  "apiProvider": "anthropic",
  "apiKey": "polza_ваш_ключ",
  "baseUrl": "https://api.polza.ai/v1"
}
```

### Для Cursor IDE
```json
{
  "anthropic.apiKey": "polza_ваш_ключ",
  "anthropic.baseUrl": "https://api.polza.ai/v1"
}
```

### Для прямого API использования (Python)
```python
import anthropic

client = anthropic.Anthropic(
    api_key="polza_ваш_ключ",
    base_url="https://api.polza.ai/v1"
)

response = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Привет!"}
    ]
)

print(response.content[0].text)
```

### Для Node.js
```javascript
import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic({
  apiKey: 'polza_ваш_ключ',
  baseURL: 'https://api.polza.ai/v1'
});

const message = await client.messages.create({
  model: 'claude-3-5-sonnet-20241022',
  max_tokens: 1024,
  messages: [
    { role: 'user', content: 'Привет!' }
  ]
});

console.log(message.content[0].text);
```

## 🛡️ Безопасность API ключей

### ✅ Правила безопасности:

1. **Никогда не публикуйте ключи в Git**
   ```bash
   # Добавьте в .gitignore:
   .env
   .env.local
   config/secrets.json
   ```

2. **Используйте переменные окружения**
   ```bash
   # .env файл
   POLZA_API_KEY=polza_ваш_ключ
   ```

3. **Ограничьте права доступа**
   - Создавайте отдельные ключи для разных проектов
   - Удаляйте неиспользуемые ключи

4. **Мониторьте расходы**
   - Проверяйте баланс в личном кабинете
   - Настройте уведомления о расходах

### ❌ Что НЕ делать:

- ❌ Не коммитьте ключи в Git
- ❌ Не делитесь ключами с другими
- ❌ Не используйте один ключ везде
- ❌ Не храните ключи в открытом виде в коде

## 📊 Мониторинг использования

### В личном кабинете Polza.ai:

1. **Баланс** - текущий остаток средств
2. **История операций** - все запросы и расходы
3. **Статистика** - графики использования
4. **Лимиты** - настройка ограничений

### Как экономить токены:

1. **Будьте конкретны в запросах**
   ```
   ❌ "Расскажи всё о Django"

   ```

2. **Используйте контекст проекта**
   - Cline видит структуру проекта
   - Не нужно повторять весь код

3. **Разбивайте большие задачи**
   ```
   ❌ "Создай полное приложение с авторизацией, API, тестами..."
   ✅ "Создай модели для приложения"
   ✅ "Добавь API эндпоинты"

   ```

## 🆘 Решение проблем

### Проблема: "Invalid API Key"
**Решение:**
- Проверьте правильность ключа
- Убедитесь, что ключ активен в личном кабинете
- Проверьте баланс

### Проблема: "Rate limit exceeded"
**Решение:**
- Подождите несколько минут
- Проверьте лимиты в настройках
- Пополните баланс

### Проблема: "Connection timeout"
**Решение:**
- Проверьте интернет-соединение
- Проверьте baseUrl в настройках
- Попробуйте позже

### Проблема: Высокий расход токенов
**Решение:**
- Делайте более конкретные запросы
- Используйте более дешевые модели для простых задач
- Проверьте историю запросов
