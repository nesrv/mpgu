# Билеты для экзамена по дисциплине "Управление разработкой программного обеспечения"


## Билет 1: Введение в управление разработкой ПО. Основные понятия и терминология

### Теоретическая часть:

1. Дайте определение управления разработкой программного обеспечения и назовите основные цели и задачи.
2. Опишите основные различия между управлением проектами и управлением разработкой ПО.

### Практическая часть:

Проанализируйте кейс неуспешного проекта разработки ПО и определите основные причины неудачи. Предложите меры по предотвращению подобных проблем в будущем.
