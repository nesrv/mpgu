# Программа экзамена: Управление разработкой программного обеспечения (2026)

## Темы экзамена

### 1. Обзор управления разработкой ПО
Определение основных понятий:
- Управление разработкой ПО
- Процессы разработки
- Методологии
- Жизненный цикл разработки
- DevOps и DevSecOps культура
- Platform Engineering

### 2. Процессы разработки ПО
Классификация и описание основных процессов:
- Анализ требований
- Проектирование
- Кодирование
- Тестирование (Unit, Integration, E2E)
- CI/CD и автоматизация развертывания
- Мониторинг и observability
- Поддержка и инциденты

### 3. Современные методологии разработки ПО
Обзор и сравнение актуальных методологий:
- Agile и его эволюция
- Scrum и Scrum@Scale
- Kanban и Scrumban
- SAFe (Scaled Agile Framework)
- LeSS (Large-Scale Scrum)
- Shape Up (Basecamp)
- Extreme Programming (XP)

### 4. Жизненный цикл разработки ПО
Основные этапы жизненного цикла:
- Discovery и Product Discovery
- Планирование и приоритизация
- Разработка (Development)
- Тестирование и QA
- Deployment и Release Management
- Мониторинг и поддержка
- Continuous Improvement

### 5. Управление требованиями к ПО
Современные подходы к работе с требованиями:
- User Stories и Job Stories
- Use Cases и User Scenarios
- Behavior-Driven Development (BDD)
- Acceptance Criteria
- Product Backlog Management
- Приоритизация (RICE, MoSCoW, Kano)

### 6. Проектирование и архитектура ПО
Современные подходы к проектированию:
- Микросервисная архитектура
- Event-Driven Architecture
- Serverless архитектура
- Domain-Driven Design (DDD)
- Clean Architecture
- API-First подход
- Архитектурные паттерны (CQRS, Event Sourcing)

### 7. Качество и тестирование ПО
Современные практики обеспечения качества:
- Test-Driven Development (TDD)
- Behavior-Driven Development (BDD)
- Автоматизация тестирования
- Shift-Left Testing
- Continuous Testing
- Performance и Load Testing
- Security Testing (SAST, DAST)
- Chaos Engineering

### 8. DevOps и CI/CD
Практики непрерывной разработки:
- Continuous Integration
- Continuous Delivery/Deployment
- Infrastructure as Code (IaC)
- GitOps
- Containerization (Docker, Kubernetes)
- Мониторинг и логирование
- Incident Management

### 9. Управление проектами и продуктами
Современные подходы к управлению:
- Product Management vs Project Management
- OKR (Objectives and Key Results)
- Roadmap Planning
- Stakeholder Management
- Metrics и KPI (DORA metrics, SPACE framework)
- Data-Driven Decision Making
- A/B тестирование и эксперименты

### 10. Командная работа и культура
Построение эффективных команд:
- Team Topologies
- Психологическая безопасность
- Remote и Hybrid работа
- Engineering Culture
- Code Review практики
- Pair Programming и Mob Programming
- Knowledge Sharing

### 11. AI и автоматизация в разработке (NEW)
Использование AI в разработке ПО:
- AI-ассистенты (GitHub Copilot, ChatGPT)
- Автоматизация code review
- AI в тестировании
- Генерация документации
- Этика использования AI в разработке

### 12. Безопасность и compliance (NEW)
Безопасность в современной разработке:
- DevSecOps практики
- Secure Software Development Lifecycle (SSDLC)
- Vulnerability Management
- Compliance и регуляторные требования (GDPR, SOC2)
